# Beat Breakdown 输出格式（参考）

**此模板仅供参考 - 最终输出不应包含这些说明**

**❌ 严禁输出**：

- Frontmatter 元数据
- 模板说明
- "下一步"指令
- 任何非内容的说明

**✅ 仅输出**：实际的 beat breakdown 内容

---

## 最终输出应包含：

```markdown
# Beat Breakdown - Episode {XX}

**场景**: [场景引用]
**描述**: [发生了什么]
**目的**: [叙事功能]

## Beat 2: [标题]

**场景**: ...
**描述**: ...
**目的**: ...

...

## Beat 9: [标题]

**场景**: ...
**描述**: ...
**目的**: ...
```

## Agent 注意事项：

- 无需 frontmatter 元数据
- 无需模板说明
- 仅输出实际 beat breakdown 内容
- 恰好 9 个 beats
- 每个 beat 包含：标题、场景、描述、目的
